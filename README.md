# git-command-Djamel

##Quelques commandes Git

>git revert : permet d'annuler un commit fait précédemment

>git diff : permet de voir les différences entre les fichiers actuels et le dernier commit

>git push : permet d'envoyer sur  GitHub

>git fetch : permet de récupérer les fichiers sur GitHub
